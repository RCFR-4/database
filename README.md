# database

Repository for database fundamentals, logical modeling, and SQL practice projects.

## Structure

- `01-logical-modeling/`: logical modeling exercises, entities, attributes, relationships, PK/FK, and problem statements.
- `02-projects/`: practical SQL projects with `schema.sql`, `seed.sql`, `queries.sql`, and project notes.

## Purpose

This repository is used to organize database study and practice, combining conceptual modeling with practical SQL implementation.
