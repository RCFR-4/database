# Problem Statement

A person wants a system to organize a personal study library.

The library owner wants to keep a record of authors, including information such as full name, nationality, birth date, death date, and a short biographical note.

The system also needs to keep a record of works, including title, description, original language, original publication year, and reading status.

Some works may have more than one author, and the same author may be related to different works.

The library owner also wants to organize works by themes. For each theme, the system should store information such as theme name and description.

A work may be related to different themes, and the same theme may be related to different works.

In addition, the system must keep a record of editions or versions of each work. Each edition belongs to a specific work and should include information such as format, publisher, edition year, and additional notes.

With this system, the library owner wants to be able to identify:
- which authors are registered
- which works are registered
- which themes are available
- which authors are related to each work
- which works are related to each author
- which themes are related to each work
- which works are related to each theme
- which editions are available for each work
- which works are not started, in progress, or completed
- which works have more than one edition or version
